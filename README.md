# Horde RL

Horde RL is a roguelike version of the classic game The Horde.


# Running the Source Code

> pipenv install
> pipenv run python ./hordeRL.py

# Roadmap

## Defend Scene

* Smarter pathfinding for hordelings
* Escape AI for peasants
* Bump attack
* Knight allies
* Houses

## Bugs

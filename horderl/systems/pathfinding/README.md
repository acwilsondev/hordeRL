# Pathfinding Systems

This directory hosts pathfinding-related systems and orchestration logic that
operate on pathfinding components and cost maps. It owns decision-making helpers
for selecting targets and coordinating pathfinding behavior across gameplay
features.

## Responsibilities
- Target selection helpers that interpret evaluated entities.
- System-side logic that coordinates pathfinding behavior.

## Feature verticals
- Target selection utilities used by brains and terrain systems.
